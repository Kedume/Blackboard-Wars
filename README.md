# Blackboard-Wars
Game for the Ludum Dare 32

The theme for the Ludum Dare 32 was "an unconventional weapon".

The idea is to make a classic arcade game into a blackboard. The unconventional weapon is a "chalk".

CONTROLS

Cursors: move
Space: fire
Alt: bomb
